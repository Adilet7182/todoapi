# TodoList Helper Bot 📝

Telegram бот для управления списком задач. Добавляйте дела, отмечайте выполненные и удаляйте — всё прямо в Telegram, всегда под рукой.

## Команды

- `/start` — начало работы и список команд
- `/add (текст)` — добавить задачу
- `/list` — показать все задачи
- `/done (номер)` — отметить задачу выполненной
- `/delete (номер)` — удалить задачу
